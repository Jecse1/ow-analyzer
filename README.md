# 오버워치 스크림 분석기

## 구동 방법 (Env: dev)

```
개발환경 PowerShell 기준으로 작성되었습니다.
```

### Backend

#### 설치

```
python -m venv backend\venv
.\backend\venv\Scripts\Activate.ps1
pip install -r backend\requirements.txt
```

#### 실행

```
python backend\main.py
```

### Frontend

#### 설치

```
cd frontend
npm install
```

#### 실행

```
npm run dev
```
